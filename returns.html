Returns

If you are not satisfied with your purchase from Varkhart (Pty) Ltd, you can return your order for a refund, minus 20% of the total purchase price for transaction and admin costs, within six (6) days from the date of purchase. The items returned must be unworn, unused, unwashed and in the original packaging with all original tags attached for the return to be accepted.

Varkhart (Pty) Ltd Support is from 9:00am to 4:00pm Monday – Friday

Once your return is processed, you will receive an email that contains your Return Reference number. Please send the parcel to the below address.

Please note the returns shipping charges are for your own account; unless the item is defective – then we will cover the costs. – 8 Reitz Street, Deneysville, Free State, 1932.

Please print the Return Reference number clearly on the outside of the box and include the printed email inside. The Return Reference number is required for us to process your return and refund. Refunds are processed when products are received.
